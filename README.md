## How To Start
- Start Game with "Main Menu" Scene

## Gameplay
- Move Camera with WASD or Arrow Keys
- Shoot NPCs with Left Mouse Button
- Change Ammo with Numbers 1,2 and 3
- Open Help with 0
- Reload with R

## Built With

- Unity
